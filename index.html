Research & Projects

4D Cardiac MRI Segmentation (ST-ACDC-Net)
Medical Image Analysis — 2026 – Present
Developing a spatio-temporal attention network for 4D cardiac MRI dynamic segmentation. Built upon 3D TransUNet and CA-Net architectures, trained on the CAMUS 4-chamber dataset. Involves multi-round training pipelines with comprehensive evaluation of segmentation metrics.
BLE-Based Real-Time Asset Tracking with LLM-Assisted Diagnosis
Final Year Project, BNBU — 2026 – Present
Designing an indoor asset tracking system using BLE RSSI fingerprinting combined with LLM-assisted signal anomaly diagnosis. The system includes real-time localization, a web dashboard for visualization, and an LLM module for calibration suggestions and natural-language explanations of tracking failures (human body blockage, multipath, beacon displacement).
Competition Arrow Performance Evaluation & Intelligent Selection
Olympic Key Research Program (China Institute of Sport Science) — June 2025 – Present
Contributing to the Olympic key project “Smart Arrow Selection Equipment and System R&D” for LA 2028 preparation. Operated high-speed camera setups for arrow release analysis, extracted kinematic factors (release angle, velocity) linked to impact scores, and applied computer vision trajectory tracking to quantify arrow straightness.
Flood Forecasting & Hydrological Modelling
UIC – Sun Yat-sen University – Huawei Collaboration — Sept 2025 – Present
Processed remote-sensing imagery for hydrological modelling and built an ML pipeline to predict station discharge from spatiotemporal features. Contributed to the land-surface runoff component and integrated data and ML outputs for flood forecasting experiments.
RGB-D Cargo Height Estimation (CGF-Height)
Mobile Deployment — 2026
Developed a mobile-friendly RGB-D depth-based cargo height estimation system using center-guided segmentation and local ground-aware fusion (CGF-Height). Designed for real-time deployment on mobile devices with a HeightScanner API.
